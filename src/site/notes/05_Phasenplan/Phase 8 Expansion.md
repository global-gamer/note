# Phase 8 – Multi-Node & Regional Expansion

**Ziele:** Infrastruktur und Community auf weitere Regionen/Games skalieren, Node03+ an Netz bringen.

## Key Workstreams
- Node03 Rollout (ggf. andere Region/Provider) + Game-Spezifika (Shooter, MOBA) vorbereiten.
- Regional Community Leads (D/A/CH) und Ambassador Programme starten.
- Multi-Lingual Support im Forum (EN-Sektion light) + SSO für Partnerplattformen.
- Partnerships auf Events/Offlines ausweiten (LAN-Präsenz, Pop-up Booths).

## Economic Focus
- ROI-Betrachtung jeder Region/Game (KPIs: ARPU, Engagement, Cost-to-Serve).
- Sponsoren-Pakete für regionale Events (Co-branding, Product Sampling).

## Check / KPIs
- Node03 produktiv, Auslastung <70 %, Monitoring/Backups analog.
- 2 neue Fokus-Games mit ≥80 aktiven Spielern.
- Regionale Ambassadors liefern monatliche Reports, Community Satisfaction >8/10.

## Milestones & Owners
| Deliverable | Owner |
| --- | --- |
| Node03 Deployment (Region/Provider) + Checklist | Operations Director |
| Regional Ambassador Program Launch | Community Director |
| Multi-Lingual Forum Section live | Community Manager |
| Partnerships for Regional Events (≥2 deals) | Partnerships Manager |
| ROI Report pro Region/Game | Finance Lead + Data Analyst |

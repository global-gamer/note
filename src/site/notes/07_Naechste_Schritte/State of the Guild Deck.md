# State of the Guild Deck – Template

Quartalsweises Update für Community, Partner und Team. Nutzt Daten aus der [[04_Infrastruktur/Data Platform\|Data Platform]], Phase-Roadmap und Playbooks.

## Struktur
1. **Intro / Manifest**
   - CEO Statement (1 Slide) – Bezug auf [[00_Meta/Manifest\|00_Meta/Manifest]].
   - Timeline-Highlight (letzte Meilensteine).
2. **Community Pulse**
   - KPIs: WAU/MAU, Event Attendance, NPS.
   - Top Member Stories / Clips.
   - Moderation & Support Stats.
3. **Growth & Monetization**
   - Revenue Run Rate, Sponsoren, Playbook Status (z. B. [[05_Phasenplan/Taschengeld Funnel\|05_Phasenplan/Taschengeld Funnel]], [[05_Phasenplan/Creator Labs Playbook\|05_Phasenplan/Creator Labs Playbook]]).
   - Upcoming campaigns.
4. **Operations & Infrastructure**
   - Server health, new nodes, incident summary ([[04_Infrastruktur/Incident Playbook\|04_Infrastruktur/Incident Playbook]]).
   - Automation/ IaC progress.
5. **Roadmap & Phasenstatus**
   - Übersicht (Phase 1–10) mit Status-Icons.
   - Lessons learned + next focus.
6. **Future Bets & Calls to Action**
   - Key projects aus [[Future Bets\|Future Bets]] / [[05_Phasenplan/Playbook Backlog\|05_Phasenplan/Playbook Backlog]].
   - Calls for volunteers, partners, support.

## Datenquellen
- Grafana / Metabase Dashboards (Screenshots).
- Meeting-Notes & Shortlist-Items.
- Creator Labs / Taschengeld Funnel Reports.
- Incident/Postmortem Summaries.

## Output
- Deck in Google Slides / Canva (Template verlinken).
- Kurzfassung als Markdown/Newsletter.
- Recording (Discord Stage / YouTube) archivieren.
